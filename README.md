# Online Book Store - End-to-End SQL Project

## 📚 Project Overview
This project is an end-to-end SQL analysis of an Online Book Store. It involves managing and extracting meaningful insights from three core datasets: Books, Customers, and Orders. The goal is to solve real-world business problems using SQL queries ranging from basic to advanced levels.

## 🗂️ Dataset Structure
The project utilizes three tables:
- **Books:** Contains information about Book ID, Title, Author, Genre, Price, and Stock.
- **Customers:** Holds customer details like Customer ID, Name, Email, and Location.
- **Orders:** Stores transaction data including Order ID, Customer ID, Book ID, Order Date, Quantity, and Total Amount.

## 🚀 Key Business Queries Solved
Some of the key questions explored in this project include:
- Finding top-selling books and authors.
- Analyzing revenue generated by different genres.
- Identifying high-value customers and their ordering patterns.
- Checking stock levels and inventory management.

## 💻 Tech Stack Used
- Database: PostgreSQL
- Tool: Data Extraction & Analysis using SQL
